# Packages

- 해당 컨벤션 문서에서 언급하는 패키지를 정리한다.
- **작업을 시작하기 전에 이 목록을 확인하고, 프로젝트에 없는 패키지는 설치한 뒤 시작한다 (필수).** 이 패키지들을 기본값으로 사용한다.
- 단, 사용자가 다른 패키지를 쓰겠다고 명시하면 그 선택을 따른다.

## 사용하는 패키지

- [es-toolkit](https://www.npmjs.com/package/es-toolkit): 함수형 프로그래밍을 위한 유틸리티 라이브러리
- [react](https://www.npmjs.com/package/react): React를 위한 라이브러리
- [react-dom](https://www.npmjs.com/package/react-dom): React DOM을 위한 라이브러리
- [react-router](https://www.npmjs.com/package/react-router): React Router를 위한 라이브러리
- [suspensive](https://www.npmjs.com/package/@suspensive/react): Suspense를 사용하기 위한 라이브러리
- [suspensive-react-query](https://www.npmjs.com/package/@suspensive/react-query): React Query를 Suspense로 사용하기 위한 라이브러리
